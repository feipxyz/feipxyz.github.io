集成方法之Adaboost

发表于 2018-05-28 | 分类于 机器学习

本文介绍集成方法中的Adaboost。

Adaboost总体思想

  1. 多专家判断。
  2. 提高那些被前一轮弱分类器误分样本的权值，降低被正确分类样本的权值。
  3. 针对多个弱分类器，AdaBoost采用加权多数表决的方法，即加大分类误差率小的弱分类器的权值，使其在分类表决中起较大的作用。
