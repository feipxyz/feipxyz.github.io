Mxnet源码解读之Dataset与DataLoader

发表于 2018-06-22 | 分类于 深度学习 ， mxnet

本文介绍mxnet中数据加载的两个类Dataset与DataLoader。

Dataset基类

  1. 所有的DataSet类继承自Dataset抽象类，且必须实现__getitem__和__len__方法。
  2. transform(self, fn, lazy=True)。在原数据集上应用fn方法得到一个新的数据集。如果lazy为False，立即返回一个新的数据集，如果lazy为True，返回的是一个含有转换方法的类，在真正使用数据的时候再进行转换。fn是转换函数，针对每一个样本。
  3. transform_first(self, fn, lazy=True)，只对第一个元素做transform操作，例如只对data做操作，而保持label不变

ImageFolderDataset派生类

  1. root是存放图片的根目录。如root/car/0001.jpg root/car/xxxa.jpg root/car/yyyb.jpg
  2. flag表示要不要转换为3通道，If 0, always convert loaded images to greyscale (1 channel). If 1, always convert loaded images to colored (3 channels).

DataLoader

  1. dataset：一个Dataset的对象。
  2. batch_size：批量大小。
  3. shuffle：是否打乱次序。
  4. last_batch：当最后一个批量大小不足batch_size时，如何处理。
