linux遇到的问题

发表于 2018-06-19 | 分类于 linux

windows与linux互传文件的便捷方法

使用winscp，具体参考Windows和linux(ubuntu)互传文件简便快捷的方法。

可能会遇到拒绝连接的情况，这说明没有建立好通信。操作系统要想使用ssh协议进行通信，就要提供ssh客户端服务和ssh服务器端服务。默认情况下，ubuntu中只提供了ssh客户端服务，而没有提供ssh服务器端服务。所以windows和它通信就不成功。我们只需要进入到linux操作系统ubuntu中安装好ssh服务器端服务就行了。 具体参考使用WinSCP软件在windows和ubuntu中进行文件传输。
