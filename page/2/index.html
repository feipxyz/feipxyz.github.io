vim使用积累

发表于 2018-06-15 | 分类于 linux

.vimrc文件

.vimrc 是控制 vim 行为的配置文件，位于 ~/.vimrc，不论 vim 窗口外观、显示字体，还是操作方式、快捷键、插件属性均可通过编辑该配置文件将 vim 调教成最适合你的编辑器。

命令模式

  1. a 光标后插入，i 光标前插入，A shift+a 行尾插入，I shift+i 行首插入，o 在光标下面插入一行，O shift+o 子啊光标上面茶一行，
  2. j 向下移动一行,k 向上移动一行，h 向左移动一个字符，l 向右移动一个字符，gg为跳到第一行行首，G shift+g 最后一行行首；^ 0 移动到行首； $ 移动到行尾；
  3. Ctrl+u 视窗往上翻半页；ctrl+d 往下翻半页； Ctrl+f 视窗下翻一页；Ctrl+b 视窗往上翻一页；Ctrl+e 视窗往下卷一行；Ctrl+y 视窗往上卷一行
